# Screenshots

Drop your app screenshots into this folder. The main project README expects these filenames:

- `home.png` — Home dashboard
- `scanner.png` — Barcode / nutrition label scanner
- `progress.png` — Progress charts
- `meal_plan.png` — Meal planner
- `fasting.png` — Intermittent fasting screen
- `onboarding.png` — Onboarding flow

Recommended size: your phone's native resolution (e.g. 1080×1920). They render at 220px wide on the README, so smaller images work too.

If you don't have all six, just remove the corresponding `<img>` tags from the main `README.md`.
